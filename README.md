# Hackton Portal

## Introduction
Hackton Portal is a web application designed to streamline the management of hackathons, providing tools for organizers and participants to interact effectively.

## Features
- User registration and management
- Event creation and scheduling
- Team collaboration features
- Real-time notifications
- Performance tracking and analytics

## Installation

To set up the Hackton Portal locally on your machine, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/RaminIsmailsoy/hackton-portal.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd hackton-portal
   ```

3. **Install the dependencies**
   ```bash
   npm install
   ```

4. **Start the application**
   ```bash
   npm start
   ```

The application should now be running on `http://localhost:3000`.

## Usage

Once the application is running, navigate to the homepage to register or log in. Organizers can create events, and participants can join and form teams.
